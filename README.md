# Hotel Gopal Midway — Premium Restaurant Website

> A world-class, feature-rich restaurant website built with React — redesigned from the ground up.

---

## 🚀 Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm start

# 3. Build for production
npm run build
```

## 🛠 Tech Stack

| Technology | Purpose |
|-----------|---------|
| React 18 | UI Framework |
| React Router DOM | Navigation (extendable) |
| Framer Motion | Animations (extendable) |
| React Icons | Icon library (extendable) |

| Intersection Observer | Scroll animations |

| Google Maps Embed | Location |
